# ConfigPilot v1.0.13

本版本统一 Codex 上下文预设，移除 GPT-5.6 的高上下文配置路径，避免中转服务因不支持对应数值而报错。

## 主要更新

- “套用上下文预设”简化为单一的“套用稳定上下文”按钮。
- GPT-5.5 与 GPT-5.6 统一使用 GPT-5.5 稳定值：上下文窗口 `258400`、自动压缩阈值 `245000`、工具输出保留 `6000`。
- 保存配置时会把旧的 GPT-5.6 `372000 / 353000` 上下文值限制到稳定范围，避免继续写入已知不兼容值。
- GPT-5.6 的推理等级选项保持不变，本次仅调整上下文预算。
- 帮助页与项目文档同步更新为统一稳定上下文说明。

## 升级说明

ConfigPilot v1.0.12 已包含应用内更新器。Windows 用户可在应用内完成下载、静默安装和重启；macOS 用户检测到新版后会打开官方 Release 页面。

升级后重新选择模型或点击“套用稳定上下文”，再点击“应用”，即可写入统一后的稳定值。

## macOS 首次安装须知

当前 macOS 版本仅支持 Apple Silicon（M 系列芯片），不支持 Intel Mac。

维护者目前没有 Apple Developer Program 会员，因此 DMG 尚未使用 Developer ID 签名和 Apple 公证。macOS Gatekeeper 可能显示“ConfigPilot 已损坏，无法打开”，但这不代表 DMG 文件真的损坏。

请先把 `ConfigPilot` 拖入 `Applications`，再打开“终端”执行：

```bash
xattr -dr com.apple.quarantine "/Applications/ConfigPilot.app"
open "/Applications/ConfigPilot.app"
```

同一已安装版本通常只需执行一次。请仅从本项目官方 Releases 下载。
